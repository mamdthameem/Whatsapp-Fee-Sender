# Assets Folder

## College Logo

Place your college logo image here as `college-logo.png`

### Requirements:
- **Filename:** `college-logo.png`
- **Format:** PNG (with transparency preferred)
- **Size:** Recommended 200x200px or larger (will be resized to 80x80px)
- **Location:** `frontend/assets/college-logo.png`

### Current Setup:
- The logo is displayed in the top-left header section
- If the logo file is not found, a fallback "SSEC" badge will be displayed
- Logo is styled with a gold border and shadow for professional appearance

### To Add Your Logo:
1. Save your college logo image as `college-logo.png`
2. Place it in this `frontend/assets/` folder
3. The logo will automatically appear in the header
